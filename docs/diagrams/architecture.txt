                   FUSION PIPELINE
                   ===============

    Results from Mem0  Cognee  Letta  Graphiti
          |             |        |        |
          v             v        v        v
    +-----------------------------------------------+
    |              Fusion Engine                      |
    |                                                |
    |  STAGE 1: Collection & Normalization           |
    |  +------------------------------------------+  |
    |  | Gather all entries from all results       |  |
    |  | Normalize relevance to [0, 1] range       |  |
    |  | normalized = entry.rel / max(all.rel)     |  |
    |  +------------------------------------------+  |
    |                      |                         |
    |                      v                         |
    |  STAGE 2: Deduplication                        |
    |  +------------------------------------------+  |
    |  | For each pair of entries:                 |  |
    |  |   if embeddings: cosine_similarity        |  |
    |  |   else: jaccard_similarity (tokens)       |  |
    |  |                                           |  |
    |  |   if sim >= 0.92: keep higher confidence  |  |
    |  +------------------------------------------+  |
    |                      |                         |
    |                      v                         |
    |  STAGE 3: Cross-Source Re-Ranking              |
    |  +------------------------------------------+  |
    |  | score = relevance * 0.40                  |  |
    |  |       + recency   * 0.25                  |  |
    |  |       + source    * 0.20                  |  |
    |  |       + type      * 0.15                  |  |
    |  |                                           |  |
    |  | recency = exp(-0.00413 * hours)           |  |
    |  | (half-life ~7 days)                       |  |
    |  |                                           |  |
    |  | Sort by score DESC, apply TopK            |  |
    |  +------------------------------------------+  |
    |                      |                         |
    +----------------------|------------------------+
                           |
                           v
                    SearchResult {
                        Entries: ranked & deduped
                        Total: count
                        Duration: aggregate
                        Sources: responding backends

                CIRCUIT BREAKER PATTERN
                =======================

    For each backend client (Mem0, Cognee, Letta, Graphiti):

                  +--------+
           +----->| CLOSED |<-----+
           |      +---+----+      |
           |          |           |
           |    failures >= 5     | 2 consecutive
           |          |           | successes
           |          v           |
           |      +--------+     |
           |      |  OPEN  |-----+
           |      +---+----+     |
           |          |          |
           |   timeout (30s)     |
           |   expires           |
           |          |          |
           |          v          |
           |    +-----------+    |
           +----| HALF-OPEN |----+
                +-----------+

    CLOSED:    All requests pass. Failures counted.
    OPEN:      All requests fail-fast. No network calls.
    HALF-OPEN: Test requests. 2 successes = close. 1 failure = reopen.


                CONSOLIDATION ENGINE
                ====================

    Background loop (default: every 30 minutes)

    +----> Sleep for ConsolidationInterval
    |           |
    |           v
    |   Phase 1: Collect
    |   - Fetch recent entries from ALL providers
    |   - Batch size: 100
    |           |
    |           v
    |   Phase 2: Deduplicate
    |   - Remove exact ID duplicates
    |           |
    |           v
    |   Phase 3: Enrich
    |   - Add consolidated_at timestamp
    |   - Add source_count metadata
    |           |
    |           v
    |   Update Stats
    |   - memories_processed
    |   - deduplicated
    |   - consolidated
    |   - errors
    |           |
    +-----------+
