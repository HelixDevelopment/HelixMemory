Intelligence Layer

Fusion Pipeline

Three precision stages transform raw multi-source memories into unified, ranked intelligence.

Stage 1

Collection & Normalization

Parallel queries across all four backends. Results normalized into a unified MemoryEntry format with source attribution, timestamps, and confidence metadata.

Stage 2

Deduplication

Cosine similarity analysis detects and merges semantically equivalent entries across sources. Threshold at 0.92 eliminates noise while preserving nuance.

Stage 3

Cross-Source Re-Ranking

A weighted scoring formula combines relevance, recency, source reliability, and memory type to produce a final ranked set of memories.

Scoring Formula
score = relevance * 0.40 + recency * 0.25 + source * 0.20 + type * 0.15
Deduplication threshold:
cosine sim = 0.92

Quick Start

Getting Started

From infrastructure to first query in under five minutes.

docker/docker-compose.yml 
# Start the full HelixMemory infrastructure
cd docker && docker compose up -d

# Services started:
#   Letta     -> :8283  (agent runtime)
#   Mem0      -> :8001  (fact extraction)
#   Cognee    -> :8000  (knowledge graphs)
#   Qdrant    -> :6333  (vector store)
#   Neo4j     -> :7474  (graph database)
#   Redis     -> :6379  (cache layer)
#   PostgreSQL -> :5432 (persistence)

# Verify all services are healthy
docker compose ps
provider/unified.go 
import (
    "digital.vasic.helixmemory/pkg/config"
    "digital.vasic.helixmemory/pkg/provider"
)

// Load configuration from environment
cfg := config.LoadFromEnv()

// Create the unified memory provider
ump, err := provider.NewUnifiedMemoryProvider(cfg)
if err != nil {
    log.Fatalf("failed to init HelixMemory: %v", err)
}

// Start background services (consolidation, metrics)
ump.Start(ctx)
defer ump.Stop()
example_usage.go 
// Store a memory (auto-routed to optimal backend)
entry := &types.MemoryEntry{
    Content:  "User prefers functional error handling",
    Type:     types.MemoryTypeFact,
    UserID:   "agent-001",
    Metadata: map[string]any{"project": "helixagent"},
}
id, err := ump.Add(ctx, entry)

// Search across all backends with fusion
results, err := ump.Search(ctx, "error handling patterns", 10)

// Results are fused, deduplicated, and ranked
for _, r := range results {
    fmt.Printf("[%.2f] %s (via %s)\n",
        r.Score, r.Content, r.Source)
}
